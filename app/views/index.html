Employer incentives design history

What is employer incentives?

We are designing and implementing the functionality to provide financial incentives to employers who newly employ apprentices aged between 16-24 from September 2020 onwards.

This functionality is part of a set of Covid-19 recovery initiatives being run by ESFA.

Design team on this project

  • Jas Johal - interaction designer
  • Oli Rose - content designer
  • Ben Towers - user researcher
